Anliegen melden

Über "Sag's uns" können Sie die Stadtverwaltung über Probleme im Kölner Stadtbild informieren. Dazu gehen Sie einfach auf "Neues Anliegen" im Menü, machen ein Bild, wählen eine passende Kategorie und senden uns ihr Anliegen. Falls gewünscht, können Sie uns in zusätzlich in wenigen Worten den Grund ihres Anliegens schildern.

Deine Anliegen anzeigen
Über das Menü können Sie jederzeit den Status ihrer Anliegen einsehen.
